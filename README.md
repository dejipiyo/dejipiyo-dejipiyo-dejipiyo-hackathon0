# rebuild-base-frontend
## Installation
### Clone
```shell
git clone git@github.com:rebuild-inc/rebuild-base-frontend.git
cd rebuild-base-frontend
```

### Copy .env.local
```shell
cp .env.example .env.local
```

### Install
```shell
yarn install
```

### Start
http://localhost:3000
```shell
yarn run dev
```

## Usage
### Lint
```shell
yarn run lint
yarn run lint:fix
yarn run type:check
```

### Format
```shell
yarn run format
yarn run format:check
```

### Test
```shell
yarn test
```

### Codegen
```shell
yarn codegen
```
